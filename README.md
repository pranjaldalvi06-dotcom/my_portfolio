# Pranjal Dalvi — Data Analyst Portfolio

## 📁 File Structure

```
pranjal-portfolio/
│
├── index.html              ← Main HTML (entry point — open this in browser)
│
├── css/
│   └── style.css           ← All styles, colors, animations, responsive
│
├── js/
│   ├── data.js             ← YOUR CONTENT: projects, skills, marquee items
│   └── main.js             ← All interactions: cursor, nav, animations, form
│
├── images/
│   └── profile_1.jpg       ← Add your photo here (4:5 ratio works best)
│
└── assets/
    └── Pranjal_Dalvi.pdf   ← Drop your resume PDF here
```

## ✏️ How to Customize

### 1. Your Projects
Open `js/data.js` → edit the `PROJECTS` array.
Each project needs: `emoji`, `bg` (CSS gradient), `title`, `desc`, `tags[]`, `demo` link, `code` link.

### 2. Your Skills
In `js/data.js` → edit `SKILLS` array. Add/remove categories and items.

### 3. Your Photo
Drop your photo named `profile_1.jpg` into the `images/` folder.

### 4. Your Resume
Drop your PDF named `Pranjal_Dalvi.pdf` into the `assets/` folder.

### 5. Contact Links
In `index.html` → find the `#contact` section and update email + LinkedIn URL.

## 🎨 Color Scheme
- Background: Deep forest `#060d0a`
- Accent: Emerald green `#10b981`
- Pop: Gold/amber `#f59e0b`
- Text gradient: Emerald → Gold

## 🚀 Running
Just open `index.html` directly in your browser. No build tools needed!
